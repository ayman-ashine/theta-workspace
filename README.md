# Theta Workspace
### Description :
> Theta Workspace is a web application dedicated to those who want to manage their time and daily tasks for more productivity and efficiency.

### Available Tools :
- [x] `Note`
- [x] `Todo List`
- [x] `Timer`
- [x] `Chrono`
- [ ] `Calendar`
- [ ] `Clip`
- [ ] `Linker`
